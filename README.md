# 🚀 7-Day LeetCode Challenge (Strings & Arrays)

This challenge focuses on solving high-frequency **interview problems** using Python.
Goal: Improve problem-solving skills, recognize patterns, and write optimized solutions.

---

## 📅 Day-wise Problems

### ✅ Day 1: Two Sum

**Pattern:** HashMap
Find two numbers such that they add up to a target.

```
Input: nums = [2,7,11,15], target = 9  
Output: [0,1]
```

---

### ✅ Day 2: Best Time to Buy and Sell Stock

**Pattern:** Greedy / Single Pass
Find maximum profit from one buy and one sell.

```
Input: [7,1,5,3,6,4]  
Output: 5
```

---

### ✅ Day 3: Contains Duplicate

**Pattern:** HashSet
Check if any element appears more than once.

```
Input: [1,2,3,1]  
Output: True
```

---

### ✅ Day 4: Product of Array Except Self

**Pattern:** Prefix Product
Return array where each element is product of all others.

```
Input: [1,2,3,4]  
Output: [24,12,8,6]
```

---

### ✅ Day 5: Maximum Subarray

**Pattern:** Kadane’s Algorithm
Find contiguous subarray with maximum sum.

```
Input: [-2,1,-3,4,-1,2,1,-5,4]  
Output: 6
```

---

### ✅ Day 6: Valid Anagram

**Pattern:** Frequency Count
Check if two strings are anagrams.

```
Input: s = "anagram", t = "nagaram"  
Output: True
```

---

### ✅ Day 7: Longest Substring Without Repeating Characters

**Pattern:** Sliding Window
Find length of longest substring without repeating characters.

```
Input: "abcabcbb"  
Output: 3
```

---

## 🎯 Rules

* Solve **1 problem per day**
* Write clean and optimized code
* Add solution in `/solutions/day-x.py`
* Mention time & space complexity
